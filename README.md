# Bangladesh Lawyer Agent Blueprint

This repository contains structured specifications for a Bangladesh-focused legal drafting agent.

## Specifications
- `spec/requirements.yaml` — target users, domains, languages, and outputs.
- `spec/sources.yaml` — authoritative sources and licensing status.
- `spec/knowledge_architecture.yaml` — template library, retrieval index, citations.
- `spec/guardrails.yaml` — jurisdiction limits, disclaimers, conflicts, privacy.
- `spec/workflows.yaml` — intake, clause selection, versioning, review.
- `spec/evaluation.yaml` — quality, compliance, and expert review checks.
- `spec/delivery.yaml` — delivery surface, audit, export, access controls.
- `spec/maintenance.yaml` — update cadence and monitoring.

These files are the implementation-ready baseline for building the agent and its governance.
